== Description ==

sciolism 2019 is a light-weighted and minimalistic theme. The theme was designed for blogs.

== Installation ==

1. Download the theme and move all files to the `wp-content/themes` folder.
2. Active the theme in your WordPress installation

== Usage ==

The usage of the theme is self-explaining. Customization of the header and footer can be done in the Customizer.

== Frequently Asked Questions ==

= Non-supported features =

If you use the theme you will see that there are currently some features that are not (yet) supported e.g.

* Support for RTL languages
* Support for custom headers and custom backgrounds
* No Widget areas

Depending on user feedback those features might be included in future releases.
